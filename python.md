# 2 Assignment operators 
```jsx
Assignment operators are used to assign values to variables
Addition(+): a + b
Subtraction(-): a - b
Multiplication(*): a * b
Division(/): a / b
Modulus(%): a % b
Floor division(//): a // b
Exponentiation(**): a ** b
```
